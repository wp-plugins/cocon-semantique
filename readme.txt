Optimisation SEO en modifiant la structure de votre blog en silos semantiques.

== Description ==

Organisation du contenu du blog en silo :

+ Liens vers les catégories principales sur la home
+ Liens vers les sous catégories ou les articles

1 widget affichant le plan du silo.

== Installation ==

1. Uploadez le dossier `cocon-semantique` dans `/wp-content/plugins/`
2. Activez le plugin depuis le menu `Extensions` de Wordpress
